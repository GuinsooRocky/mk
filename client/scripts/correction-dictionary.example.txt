# MK 纠错字典 — .txt 格式
#
# 用法：拷到 ~/.we/correction-dictionary.txt，然后在 ~/.we/config.json 启用：
#   "polish": {
#     "context_dictionary_enabled": true,
#     "context_dictionary_path": "~/.we/correction-dictionary.txt"
#   }
#
# 规则：
#   - 一行一词
#   - # 开头是注释
#   - 空行忽略
#   - 总数 Apple 建议 ≤ 100
#
# 反向纠错（C2）：用 `|` 分隔挂"错音"，识别后自动替换为正字
#   语法：  正字 | 错音1 | 错音2 ...
#   示例：  SVG | SAG | S A G        ← 听到 SAG/S A G 自动改回 SVG
#          SwiftUI | Swift U I | swift you ai
#   原理：长错音优先匹配，避免短串覆盖长串
#
# 比 JSON 格式更易编辑；如要保留 errors/frequency 元信息用 .json 版本
